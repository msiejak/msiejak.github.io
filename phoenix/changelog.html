Release 1.5 Phoenix

This update was released on 12/12/2020

UI/UX Changes

  • Home: Replaced the dismiss button with an "X" button to the bottom of the changelog sheet, along with an option to open this page
  • Settings: Contact button is now a contact image button
  • Increased text size (not for titles or headings)
  • Instant Run: Black cursor, underline for the text input dialog
  • Instant Run: Text hint in the input dialog
  • Settings: Added button to view the full changelog here
  • Themes: New theme "System Default". This theme responds the the system theme and uses a light theme or dark theme accordingly
  • Messages: Consistent colors for buttons; Green for bottom bars and orange for popup windows
  • Chromebooks/Large Tablets: Added different grid-based layouts for home, topics, settings, the Hidden Activity, and the "Completed Course" screen
  • Chromebooks/Large Tablets: The text size will increase on bigger devices when in landscape orientation
  • Chromebooks/Large Tablets, 1st Module: The CrOS Instructions are automatically expanded
  • Chromebooks/Large Tablets, Home: The courses are now numerically numbered
  • Third-Party Links: An integrated Chrome tab inside the app replaced integrated WebViews for some links
  • Hidden Activity: When you copy a command, it will use a snackbar instead of a toast message (now consistent with the rest of the app)
  • Topics: "Setup" is now the first topic
  • Settings: Updated switch colors

Bugfixes

  • Themes: All items now respond to themes and are properly colored
  • Transitions: All items now have transition animations
  • Chromebooks: The top window bar is now properly colored
  • Theme, "Rainy Day": Fixed an issue
  • Settings: Fixed a possible text overlap on image buttons
  • Content: Fixed spelling errors
  • Settings: Fixed a possible crash when trying to contact the developer
  • Hidden Activity: The commands now respect the "Reverse Command Actions" setting
  • Accessibility: Added Missing Content Descriptions on Image Buttons
  • Themes: Due to a bug with ads, themes will be unlocked for all users until this problem is resolved This problem was resolved in 1.6

New

  • "Course Completed" Screen: If you did not go through all of the modules, it will ask you to go back and complete the rest
  • Launcher Shortcuts: New launcher shortcut for "Topics"
  • Content: Added instructions to enable wireless debugging on Android R (11) and higher
  • Content: Added instructions to enable debugging natively on Chromebooks

Other

  • Redesigned Launcher Shortcuts
  • Performance Improvements
  • Changed Some Text
  • HUGE Behind the Scenes Changes
  • Replaced Links to Our Old Website With This One
  • Longer Transition Animations
  • Clarified Instructions for Using ADB From Chrome OS
